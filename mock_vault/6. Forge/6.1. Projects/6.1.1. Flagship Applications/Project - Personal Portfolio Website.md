## Goal
Build a high-performance, aesthetically stunning personal website to serve as a central hub for professional portfolio evidence, creative art showcase, and long-form writing.

## Background
As I pivot deeper into agentic AI and technical engineering, having a "living resume" that demonstrates both technical mastery and creative depth is essential. This site will replace static PDFs where possible and provide a dynamic interface for showing off "Nexus" and other flagship projects.

## Execution Plan

### Phase 1: Identity & Design
- [ ] Define the visual language (e.g., Sleek Dark Mode, Glassmorphism, Neon Accents).
- [ ] Select typography (e.g., Inter, Outfit, or Mono-space fonts for a "hacker" aesthetic).
- [ ] Create a mood board or Figma mockup for the landing page.

### Phase 2: Tech Stack & Infrastructure
- [ ] **Framework:** Next.js (App Router) for SEO, performance, and React ecosystem.
- [ ] **Styling:** Tailwind CSS or Framer Motion for high-end animations.
- [ ] **CMS/Content:** Markdown/MDX for blog posts and project case studies (keep it git-based).
- [ ] **Deployment:** Vercel for seamless CI/CD.

### Phase 3: Content Modules
- [ ] **Portfolio:** Grid of "Flagship" projects with deep-dive case studies.
- [ ] **Art Gallery:** Visual grid for digital and physical art pieces.
- [ ] **Writing:** Minimalist blog for thoughts on AI, philosophy, and engineering.
- [ ] **The "Brain" Feed:** Optional dynamic link to sanitized Vault exports or public Zettelkasten.

## Tasks

- [ ] Initialize the Next.js repository.
- [ ] Implement a "Bento Box" style layout for the landing page.
- [ ] Integrate a responsive image gallery for art.

- [ ] Create a "Contact" section or link to LinkedIn/GitHub.
