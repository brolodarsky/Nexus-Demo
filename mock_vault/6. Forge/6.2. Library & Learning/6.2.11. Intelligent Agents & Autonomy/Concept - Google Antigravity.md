Google's **Antigravity** is an IDE-native agentic coding assistant designed by the Google DeepMind team working on Advanced Agentic Coding. It is built to operate as a collaborative pair programming partner, capable of managing workspaces, writing and modifying code, executing commands, and navigating local knowledge bases.

## Core Concepts & Focus Areas

*   **IDE-Native Integration:** Antigravity operates directly within the developer's workspace, maintaining awareness of active files, cursor positions, and project structures.
*   **Agentic Capabilities:** Moves beyond basic chat interfaces by utilizing tool calls (like reading/writing files, executing commands, and interacting with MCP servers) to achieve multi-step development goals autonomously.
*   **Scientific & Advanced Workflows:** Focuses on high-complexity coding tasks, software engineering practices, and scientific coding applications.

## Key Learning Areas & Experiments

*   **System Prompt & Rules Verification:** Analyzing how Antigravity handles workspace rules and custom instructions (such as the `AGENTS.md` constitution).
*   **Tool Execution & Sandbox Boundaries:** Studying the range of tools available to the assistant (e.g. `run_command`, `multi_replace_file_content`, `view_file`) and how it operates in local environments.
*   **Comparison to other agents:** Benchmarking Antigravity against terminal-based tools like Claude Code and IDE extensions like Cursor or Cline.
