## Learning Queue — Agentic Coding Tools

A capture of agentic coding tools to learn and evaluate for integration into the workflow.

---

## 1. Claude Code
- **What it is:** Anthropic's CLI-based agentic coding tool. Runs directly in the terminal; can read/write files, run commands, and make multi-step code changes autonomously.
- **Why learn it:** Directly relevant to the Nexus agentic infrastructure and day-to-day development.
- **Resources:**
  - [Claude Code Docs](https://docs.anthropic.com/en/docs/claude-code/overview)
  - [Claude Code GitHub](https://github.com/anthropics/claude-code)
- **Status:** 🔲 To Learn

## 2. Claude Cowork (Projects)
- **What it is:** Anthropic's collaborative workspace feature — allows persistent project context, file attachments, and shared instructions across conversations.
- **Why learn it:** Useful for long-running projects like Nexus and Career Strategy where persistent context matters.
- **Resources:**
  - [Anthropic Projects Docs](https://www.anthropic.com/news/projects)
- **Status:** 🔲 To Learn

## 3. OpenAI Codex (CLI)
- **What it is:** OpenAI's agentic coding agent, designed to autonomously write, test, and fix code via natural language in a sandboxed environment.
- **Why learn it:** Benchmark against Claude Code; relevant for multi-model agentic strategy and employer skill requirements.
- **Resources:**
  - [OpenAI Codex CLI GitHub](https://github.com/openai/codex)
- **Status:** 🔲 To Learn

## 4. Google Antigravity
- **What it is:** Google DeepMind's IDE-native agentic coding assistant designed for advanced agentic coding.
- **Why learn it:** Currently active pair programming assistant for this workspace; critical for understanding native agent environments, prompt execution, and capability bounds.
- **Resources:**
  - [[Concept - Google Antigravity]] (Local Study Note)
  - [Antigravity Docs Home](https://antigravity.google/docs/home)
- **Status:** 🔲 In Progress / Actively Using
