We're always on the lookout for talented people who are motivated by meaningful work and passionate about the clean energy industry.

###### Software Engineering

## Full Stack Software Engineer

New York

##### Responsibilities

- **Day to day feature software development across the full application stack**
- **Continued development of our API layer including documentation of it**
- **Front-end development of web pages related to enrollment and customer experience**
- **Working with third-party platforms like Stripe / AWS / DocuSign and others**
- **Requirements gathering and application design across the full software development lifecycle**
- **Performance enhancements across our full application stack**

##### Qualifications

- **Mission driven, with a desire to help accelerate the transition to a clean energy economy**
- **Demonstrate-able expertise with Java and Spring Framework is very important**
- **Only candidates with computer science or engineering degrees**
- **Demonstrate-able web development expertise**
- **Strong computer science fundamentals. Understanding of data structures and software patterns**
- **Experience developing Java based web applications is required**
- **Demonstrated ability to code in JQuery, Javascript, CSS, HTML**
- **Experience and strong understanding of Ajax, Json and Rest**
- **Familiarity with Linux**
- **Experience working with the Spring Framework**
- **Complete comfort and ability with either Eclipse or IntelliJ (preferably IntelliJ)**
- **Experience working with version control and source code management tools like Github**
- **Strong oral and written communication skills**
