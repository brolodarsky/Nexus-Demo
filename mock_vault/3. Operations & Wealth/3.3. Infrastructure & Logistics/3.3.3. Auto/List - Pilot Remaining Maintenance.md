# Pilot Remaining Maintenance List
## 1. Safety & Electrical (Critical Priority)
- [ ] **[[Project - Pilot SRS Codes|Airbag Light Diagnosis]]:** Troubleshooting and repair plan for Pilot airbag system codes (61-10, 85-64, 81-43).
- [ ] **TPMS Sensor (Rear Driver):** Replace the sensor inside the rear driver-side wheel and re-program.
- [ ] **Dashcam Power Restoration:** Fix dashcams not working when the car is off (Hardwire kit or fuse tap troubleshooting).
- [ ] **Headlight Restoration:** Sand, polish, and UV seal the headlight lenses to restore clarity.
## 2. Engine & Cooling
- [ ] **New Radiator:** Replace aging plastic-tank OEM radiator with new unit to prevent "strawberry milkshake" (transmission fluid/coolant mix) failure.
- [ ] **Tensioner / Alternator (Serpentine) Belt:** Replace the outer drive belt. Inspect tensioner pulley for bearing noise.
- [ ] **Rear Valve Cover Gaskets & Tube Seals:** (The hard one). Requires clearing PS hoses and harness.
- [ ] **EGR Ports Cleaning:** Scrape out the carbon in the intake plenum passages to restore idle quality.
- [ ] **Engine Code P0430:** Catalyst System Efficiency Below Threshold (Bank 2). Suggests a failing catalytic converter or rear O2 sensor.
	- On your Honda J35 engine, **Bank 2 is the front bank** (closest to the radiator).
	- **The Diagnostic Logic:** Now that you have a new engine harness, spark plugs, and coils, the engine is finally running "clean." If the P0430 code is still there after a few hundred miles, it means the catalytic converter is physically degraded.
	- **Why it happened:** All those months of running with a bad knock sensor or stuck thermostat (running too "rich") likely overheated the "honeycomb" inside the front cat, causing it to lose efficiency.
	- **The Test:** Before replacing the expensive cat, have your mechanic check the **downstream O2 sensor** readings on his scanner. If the sensor is lazy, it might be lying about the cat's health.
## 3. Suspension & Chassis
- [ ] **Check Subframe Condition:** Inspect for structural rust, specifically at the mounting points (New Jersey/salt state precaution).
- [ ] **Check Compliance Bushings:** Inspect the large front lower control arm bushings for tearing/leaking fluid (extremely common failure on this chassis).
