# Project - Reorganizing Closets

> [!abstract] Overview
> A systematic optimization of all household closet spaces (bedroom, linen, utility). This project focuses on increasing storage density while maintaining a premium, high-end retail aesthetic. Designs and spatial layouts will be heavily curated using AI image generation and spatial text analysis.

## 🎯 Primary Objectives
- **Efficiency:** Maximizing every cubic inch of storage space; implementing "Life-Based" categorization.
- **Cost:** Balancing high-quality hardware (rods, shelving) with cost-effective organizational bins.
- **Aesthetics:** Transforming closets into orderly "showcases" that reduce mental load.

---

## 🤖 AI-Augmented Closet Design
- **Visual Reference:** Generating images of "Perfect Luxury Closets" to use as a 북극성 (North Star) for the aesthetic direction.
- **Spatial Optimization:** Describing closet dimensions to AI and providing an inventory list to receive optimized custom-shelving layout plans.
- **Inventory Logic:** Using AI to categorize items by frequency of use (Daily, Weekly, Seasonal) to determine the "Height Hierarchy" of storage.

---

## 🏗️ Closet Inventory & Zone Planning
- **Bedroom Closets:** Focus on color-coding, uniform hangers, and accessory display.
- **Linen Closets:** Focus on "Bundle Storage" and airtight preservation for seasonal items.
- **Utility Closets:** Functional accessibility for high-use tools and cleaning supplies.
