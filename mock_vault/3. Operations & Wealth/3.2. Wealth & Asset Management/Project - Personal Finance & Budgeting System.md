# Project - Personal Finance & Budgeting System

## Goal
Build a robust, unified system for tracking income, categorized spending, and recurring obligations to ensure long-term wealth management and financial transparency.

## Background
The current financial management is fragmented across multiple bank accounts and notes. There's a need for a "single source of truth" for budgeting and expense categorization (the "whole nine").

## Execution Plan

### Phase 1: Infrastructure & Audit
- [ ] Audit all active bank accounts, credit cards, and investment accounts.
- [ ] Centralize all one-time and recurring income sources.
- [ ] Review current [[Recurring Payments]] to ensure all automated bills are documented.

### Phase 2: Budgeting Framework
- [ ] Define spending categories (e.g., Housing, Transport, Food, Entertainment, Health).
- [ ] Set "North Star" savings goals and monthly spending limits.
- [ ] Decide on a tracking method (e.g., spreadsheet, specialized app, or Obsidian-native ledger).

### Phase 3: Automation & Reporting
- [ ] Implement a system for tracking "money spent types" (e.g., categorizing every transaction).
- [ ] Set up a monthly "Financial Health Review" protocol.
- [ ] Integrate with existing [[Master Framework]] and [[High Yield Savings Deep Dive]].

## Tasks
- [ ] Audit all active bank accounts and credit cards.
- [ ] List all monthly income sources.
- [ ] Categorize common expense types.
- [ ] Link existing [[Recurring Payments]] to the new budgeting system.
- [ ] Choose a primary tool for tracking (Spreadsheet vs. App vs. Obsidian).
