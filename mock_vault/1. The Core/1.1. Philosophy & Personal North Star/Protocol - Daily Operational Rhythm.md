# Protocol - Daily Operational Rhythm
**Objective:** Unified daily cadence. Balance financial stability, physical maintenance, and the momentum required specifically for the agentic career pivot.
## 1. Morning Boot Sequence (Pre-Flight)
- [ ] Read [[Goals]] & [[Current Learning]], then identify today's Pivot, Lab, and Forge tasks.
- [ ] **Finalize:** Place the 2 jobs, 1 learning module, and 1 project task on a sticky note. Close all irrelevant tabs and start.

## 2. Monday - Friday "Grind & Growth" Stack
Use this rhythm to ensure the tasks you "hate" get done during Peak CPU.

| Time Slot               | Action                                                                                                                      |
| :---------------------- | :-------------------------------------------------------------------------------------------------------------------------- |
| **8:20 AM - 8:30 AM**   | **Morning Boot Sequence:** Load North Star context into RAM.                                                                |
| **8:30 AM - 9:45 AM**   | **Career: DSA & Interview Prep:** LeetCode, system design, behavioral narratives. Sharpen the blade while the CPU is fresh. |
| **10 AM - 11:30 AM**    | **Deep Work 1: Portfolio Engineering**                                                                                      |
| **11:30 AM - 12:00 PM** | **Feed 1:** High-protein, minimum fat.                                                                                      |
| **12 PM - 1 PM**        | **Career: Applications** Apply to 2 targeted Agentic/AI Roles + CRM Sweep.                                                  |
| **1 PM - 3:30 PM**      | **Physical Exercise (2h):** Includes driving to/from gym or longer cardio sessions.                                         |
| **3:30 PM - 4 PM**      | Shower + Quick Reset before Feed 2.                                                                                         |
| **4:00 PM - 4:45 PM**   | **Feed 2:** Second high-protein session.                                                                                    |
| **5 PM - 7 PM**         | **Deep Work 2**                                                                                                             |
| **7 PM - 8 PM**         | **Cognitive Reset:** Zero-input time. No screens.                                                                           |
| **8 PM - 10 PM**        | **Deep Work 3:** Final execution & documentation.                                                                           |
| **10:30 PM - 12:00 AM** | Execute [[Protocol - Nightly System Shutdown]].                                                                             |

## 3. Weekend Cadence: "The Buffer"
| Day          | Objective                                                                                                     |
| :----------- | :------------------------------------------------------------------------------------------------------------ |
| **Saturday** | **Social Reset:** Rest, friends, go out. Protect this time — it's how the system doesn't crash.          |
| **Sunday**   | **Audit:** Run [[Protocol - Brain Maintenance]] + [[Protocol - Career Maintenance]].                        |

## 4. Core Principles
1. **Willpower Allocation:** Spend "Exhaustible Willpower" (AI Job/Apps/Targeted Learning) in the morning and early afternoon. Use "Inexhaustible Interest" (Portfolio Engineering) in the late afternoon/night.
2. **The 2h Reset:** The exercise block (including travel) is the "Firewall" between the job you hate and the projects you love. Do not skip it; it clears the "RAM Tax" from the morning.
3. **No Context Switching:** Do not check Discord/Social/Projects during the 8:30 AM - 10:30 AM DSA block. Clear cache only during the Exercise Reset.
